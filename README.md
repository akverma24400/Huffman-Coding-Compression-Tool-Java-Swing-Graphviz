📦 Huffman Coding & Decoding System
📌 Project Overview

This project implements the Huffman Coding Algorithm, a lossless data compression technique used to reduce the size of text files by assigning variable-length binary codes to characters based on their frequency.

The system performs:

✅ Text Compression (Encoding)

✅ Text Decompression (Decoding)

✅ Frequency Analysis

✅ Huffman Tree Construction

✅ Binary Code Generation
